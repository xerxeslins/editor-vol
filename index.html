Guia de Uso

Ajudante do criador de conteúdo para Viva o Linux.
Versão Beta. Serve para ajudar a criar dicas ou páginas de artigos.

Atalhos de Teclado (na caixa de texto):

  • Shift + B: Negrito
  • Shift + I: Itálico
  • Shift + K: Tecla (Kbd)
  • Shift + E: Ênfase
  • Shift + L: Link

Blocos Principais:

  • Título: Cria um título principal.
  • Texto: Parágrafos comuns. Para criar novos parágrafos no mesmo bloco, basta dar dois "Enter".
  • Lista: Cria lista com marcadores.
  • Comando: Destaque visual azulado.
  • Saída Cmd: Caixa branca para retorno do terminal.
  • Configuração: Para arquivos .conf, .ini, etc (Fundo claro).
  • Código: Bloco escuro para scripts e códigos fontes.
  • Vídeo / Imagem: Placeholders para multimídia.

Use os botões "Subir", "Descer" e "Duplicar" (ícone de folha) no topo de cada bloco para organizar.
